PyBREAKOUT
=======

Yet another Breakout game written in Python.

Requirements
------------

* Python >= 3.7, < 3.8

Installing
----------

Use ``pip`` to download and install the game. ::

    $ pip install pybreakout

Running
-------

Just type ``pybreakout`` to run the game. ::

    $ pybreakout

Controls
--------

Use the arrow keys to control the paddle.
